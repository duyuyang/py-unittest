### Challenge 2

You are given a string . Your task is to swap cases. In other words, convert all lowercase letters to uppercase letters and vice versa.

You are required to write the test case first.

#### Sample Input
```
Www.AUSPost.COM.au
```

#### Sample Output
```
wWW.auspOST.com.AU
```

#### Concept
```
my_str = 'sCe'
my_str.swapcase()
```
